# Quick Start

## Installation (einmalig):
```bash
cd /mnt/y/appdata/
git clone https://github.com/7blacky7/Test_CC_on_Unraid.git claude-code
cd claude-code
```

---

## 3 Schritte zu Claude Code auf Unraid:

### 1. Setup starten
```bash
./start.sh
```

### 2. Web-Terminal öffnen
```
http://<deine-unraid-ip>:7681
```
Login: `claude` / `claude123`

### 3. Mit Claude Max anmelden
Im Terminal:
```bash
claude-code auth login
```
→ Link im Browser öffnen → Mit Claude Max anmelden → Fertig!

---

## Jetzt loslegen:
```bash
claude-code
```

---

## Nützliche Befehle:

```bash
./start.sh    # Container starten
./stop.sh     # Container stoppen
./status.sh   # Status prüfen
./logs.sh     # Logs ansehen
```
